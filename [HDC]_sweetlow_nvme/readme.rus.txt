Драйвер NVMe для Windows 9x.

База: nvme2k 1.0.0.2 https://github.com/techomancer/nvme2k
Изменения: порт для Windows 9x, 64-битный LBA и исправления

1. Универсальные доработки
- добавлена обработка команд SCSI с использованием 64-битного LBA
  (поддержка больших приводов). Протестировано ТОЛЬКО на Windows 9x
- исправлен код завершения работы контроллера
- ускорено помещение команд в очередь и уменьшено потребление процессора
  при этом
- исправлено ненормально медленное исполнение запросов большого размера
  или ложно-позитивные ошибки при их исполнении
- почищены различные шероховатости в коде

2. Специфические доработки для Windows 9x
- порт с 2 исправлениями ошибок
- написан скрипт установки nvme9x.inf

Установка и удаление производятся как обычное оборудование,
без каких-либо специфических требований.
