 Introduction
 ============
        
    The DFE-580TX Quad Channel Server Card will provide high performance solution
    and safe the server PCI bus slot isn't enough problem. Each can provide 10M 
    or 100M half/full duplex connection to Switch. The ports running in 10M 
    or 100M wouldn't share one collision domain. These 4 ports can aggregate
    the throughput to 800Mbps at full-duplex mode. The built-in driver-level
    software can provide two feature for Sever connection- Fault-Tolerance 
    and Load-Balance. This software provides dynamic fail over and allows 
    customers to load balance network traffic across multiple network ports
    in this card.  It ensures that users maintain non-stop access to key 
    resources on the network, even if one or more of the network interface
    connections goes down. If a connection to an port is lost, this software
    will instantly take the fault-port out of the array and balance the 
    traffic across the remaining ports with no loss of data and, just as 
    importantly, without loss of connection. Also, when an adapter with 
    an NDIS 4.0 driver is brought back online, built-in software will 
    instantaneously add the adapter into the array and redistribute the 
    traffic.
    
    You can get latest releases of drivers from D-Link's Web site 
    (http://www.dlink.com). We hope our new readmes and installations
    will minimize the users effort to make our products work. Should there
    be further recommendations please contact the support manager at the
    local D-Link Office listed bellow.
