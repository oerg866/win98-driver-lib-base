 Introduction
 ------------
    Thank you for purchasing the D-Link DGE-550T Gigabit Ethernet 
    adapter.This driver support for the D-Link DGE-550T Gigabit 
    Ethernet adapter that including:

    DGE-550T Gigabit Ethernet Adapter(support 10M/100M/1000M PCI)
       
    This driver contains many kinds of Network Operating System
    installation .  Please refer to the readme file or user guide
    if any problems for installation, or please freely contact
    our technical support.
        
    You can get latest releases of drivers from D-Link's Web site
    (http://www.dlink.com). We hope our new readmes and installations
    will minimize the users effort to make our products work. Should there
    be further recommendations please contact the support manager at the
    local D-Link Office listed bellow.
